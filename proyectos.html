Mis Proyectos

Aplicación de Planificación de Bodas de Perros (TFG)

Aplicación móvil nativa para Android, desarrollada en Java, diseñada para la planificación de bodas entre perros. Este proyecto forma parte de nuestro Trabajo de Fin de Grado (TFG), completado con éxito como parte de nuestros estudios.

  • Tecnologias: Android Studio, Java, XML, APIs de Android

Este fue un proyecto colaborativo, donde mi compañero y yo realizamos todas las etapas del desarrollo. La experiencia nos proporcionó una visión completa del ciclo de vida de un proyecto de software.

⚠️ Nota: La aplicación no está publicada en tiendas. Puedes ver capturas de pantalla de su funcionamiento.

Sistema de Visualización de Datos Clínicos

Aplicación de escritorio desarrollada en Java para el departamento de nefrología de un hospital. Permite al personal médico la consulta y organización segura de datos de pacientes, diagnósticos y tratamientos.

  • Tecnologias: Java, Swing, SQL, JDBC
